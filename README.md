# Семинар на тему: Указатели на функции и статические переменные

Cеминар рассчитан на 30 минут для студентов 2-го года изучения С++. 
Он нацелен на закрепление ранее изученных тем по указателям на функции и статическим переменным, на развитие практических навыков их использования в реальных задачах.
Задание расположено в файле **task.MD** настоящего репозитория.

## Цели семинара:

- Углубить понимание работы с указателями на функции.
- Научить правильно использовать статические переменные для сохранения состояния между вызовами функций.
- Закрепить знание механизмов управления памятью и взаимодействия с кодом.

## Краткое содержание:
Задание расположено в файле task.MD, и представляет собой практическую задачу, для решения которой студенты должны будут применить изученные концепции
связанные с динамическим изменением поведения программы через использование указателей на функции. Также важную роль в задаче играют статические переменные, которые будут хранить состояние программы и обеспечивать устойчивость данных между вызовами функций.

## Особенности семинара:
- **Указатели на функции**: Студенты научатся передавать функции в качестве параметров и использовать их для гибкого управления логикой программы.
- **Статические переменные**: Будет продемонстрировано, как эти переменные могут использоваться для хранения состояний и данных между вызовами функций, а также для управления памятью более эффективно.
